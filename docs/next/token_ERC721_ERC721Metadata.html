    Internal function to burn a specific token Reverts if the token does not exist Deprecated, use _burn(uint256) instead.

    Parameters:
    owner - owner of the token to burn
    tokenId - uint256 ID of the token being burned by the msg.sender

    function _setTokenURI(uint256 tokenId, string uri) internal

    Internal function to set the token URI for a given token Reverts if the token ID does not exist.

    Parameters:
    tokenId - uint256 ID of the token to set its URI
    uri - string URI to assign
