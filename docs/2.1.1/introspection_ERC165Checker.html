Functions

  • _supportsAllInterfaces

    function _supportsAllInterfaces(address account, bytes4[] interfaceIds) internal view returns (bool)

    Interface identification is specified in ERC-165., Query if a contract implements interfaces, also checks support of ERC165.

    Parameters:
    account - The address of the contract to query for support of an interface
    interfaceIds - A list of interface identifiers, as specified in ERC-165
    Returns:
    true if the contract at account indicates support all interfaces in the interfaceIds list, false otherwise

    function _supportsInterface(address account, bytes4 interfaceId) internal view returns (bool)

    Parameters:
    account - The address of the contract to query for support of an interface
    interfaceId - The interface identifier, as specified in ERC-165
    Returns:
    true if the contract at account indicates support of the interface with identifier interfaceId, false otherwise
