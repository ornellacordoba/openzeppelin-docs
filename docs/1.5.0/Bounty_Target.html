Functions

  • checkInvariant

    abstract function checkInvariant() public returns (bool)

    Checks all values a contract assumes to be true all the time. If this function returns false, the contract is broken in some way and is in an inconsistent state. In order to win the bounty, security researchers will try to cause this broken state.

    Returns:
    True if all invariant values are correct, false otherwise.
